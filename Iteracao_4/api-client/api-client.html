Cliente API MySNiPs

Este é um arquivo de exemplo para os laboratórios parceiros da MySNiPs entenderem como usar a API de envio de genomas para serem processados e transformados em relatórios.

Autenticação na API

Deve ser feita para poder adquirir um token de autenticação para as demais requisições.

$ curl -X POST -H "Content-Type: application/json" -d "identifier=...&password=..."

Upload de genoma

É necessário informar um identificador de até 7 dígitos e enviar o arquivo de genoma no padrão abaixo (separado por tabs):

rs12564807 1 734462 AA rs3131972 1 752721 GG rs148828841 1 760998 CC rs12124819 1 776546 AA (...)
